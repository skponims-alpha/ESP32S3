# ESP32 GSM + LoRa Environmental Monitoring Node (Updated)

This repository now includes the user's full ESP32 project code in `src/main.ino`.
- MPU6050, BMP180, BH1750, DHT22, Soil sensors, LoRa SX1278, SIM7670 GSM support
- Deep sleep and interrupt-based wakeups
- Rain gauge tipping counter and GNSS via SIM module

See `/docs` for calibration and solar docs and `/hardware/pcb_gerbers` for PCB files.
